Resounding is the evolution of sound in Minecraft. It aims to build on previous attempts with enhanced algorithms and extra features, for a more performant and immersive audio experience. Plus, Resounding is designed with real audio physics in mind, to construct a more realistic audio landscape for the ultimate immersion. This mod is built to be a client-side Fabric mod. Every part of the mod is fully configurable, and it is all is completely compatible with vanilla gameplay, including custom resource packs.

NOTE: Resounding is currently in ALPHA. Critical bugs and missing features are expected. For more information see this page
